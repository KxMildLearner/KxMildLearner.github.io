Portfolio

Sky Tower

--Final Year Project--

Game : https://kxuan98.itch.io/sky-tower
Gitlab Source Code : CLICK HERE
Role : Game Designer, Programmer, Animator, Audio Engineer

Sky Tower is the idea for my Final Year Project. It's a puzzle-platforming game that aims to improve problem solving and critical thinking skills for elementary school students. The project is done individually except for the art assets and sprites. To see the detailed information on the game, please visit the Itch.io page, you can also download the game and try it yourself 😀.

The project is developed in Unity, and several tools are also downloaded to help develop the game in a much easier fashion. The scope that I have done in this game including :

  1.   Programming : The functional requirements are fully coded from scratch by me, including movement, monster detection, control monster, scene transition and loading, item interaction etc.
  2.   Visual : Using the Universal Renderer Pipeline (URP) from Unity, the 2D visual graphics can be enhanced tremendously using graphic shaders and 2D lights. 2D Sprite Editor is also used to conveniently animate characters and environment by using its separated layers.
  3.   Audio : FMOD, a sound effects engine and authoring tool for video games, it has integrated version for Unity. All music and SFX are built using FMOD.

Monster Frontier Rally

--Brakeys Game Jam 2021--

Game : CLICK HERE
Review Page : CLICK HERE
Role : Game Designer, Main Programmer

This game was created during the Brackeys Game Jam 2021, with the theme of "Stronger Together". In this project, I was the main programmer as well as the game designer throughout the development process. Monster Frontier Rally is a 2D-top down shooter game, players will have to fend off 15 waves of enemies with a random class of character given to you at the start of the game. As the game progresses, players can recruit new crewmates to join the team and defend together.

There are 4 character classes, Archer, Magician, Heavy Gunner and Rogue, each with their own strengths and weaknesses. There is more incentive in recruiting a more diverse team as the combination of different classes will let the characters receive various types of buff. For example, if your team has an archer and a magician, you will receive attack speed buff from archer and damage buff from magician.

This is the first game jam that I have joined.

Submersio

--HPS1 Summer of Shivers--

Game : CLICK HERE
Review Page : CLICK HERE
Role : Main Programmer, Audio Engineer

Another game jam creation from the HPS1 Summer of Shivers Game Jam, the goal is to create a game that has PS1 game style and the theme is Submechanophobia. This is our first time attempting a horror-based game and tried our best to capture the PS1 game feel. I try focus on audio engineering but still a core programmer during development.

The player is stuck on a sinking ship, and they need to quickly find a way out. Throughout the game environment, you can find clues here and there to piece together of what happened while you were sleeping. The game have 2 endings, what ending you got depends on you.
